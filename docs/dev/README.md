# Developer Docs

This section is the **developer-facing** documentation for Open Nexus OS.

Goals:

- fast onboarding (copy/paste examples),
- stable contracts (determinism, boundedness, purity rules),
- cross-device UX guidance (desktop/tablet/phone share one design language),
- “proof-first”: docs link to tests/goldens/markers where possible.

Entry points:

- DSL: `docs/dev/dsl/`
- UI: `docs/dev/ui/`
- Platform (SDK + keystones): `docs/dev/platform/keystone-map.md`
