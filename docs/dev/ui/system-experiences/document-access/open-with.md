# Open With

Open With is the system-owned default-handler and target-selection flow for opening content in another app.

Primary anchors:

- `tasks/TASK-0083-ui-v11c-document-picker-open-save-openwith.md`
- `tasks/TRACK-SYSTEM-DELEGATION.md`

Posture:

- target choice belongs to SystemUI,
- matching remains typed/MIME-driven,
- and the hand-off uses grants rather than ambient file authority.
