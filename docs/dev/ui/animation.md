# Animation

Animation should be:

- deterministic for goldens (where asserted),
- bounded in work per frame,
- accessible (reduced motion settings where applicable).

See also:

- `docs/dev/ui/performance-philosophy.md` for the default frame-budget/degrade stance,
- `docs/dev/ui/testing.md` for how animation scenes should be exercised.
