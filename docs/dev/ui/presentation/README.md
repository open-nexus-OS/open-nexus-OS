# Presentation

Presentation docs cover how UI surfaces appear, enter, leave, and temporarily take focus.

Use this category for:

- dialogs, sheets, popovers, toast, overlays,
- presentation transitions and lifecycle posture,
- print/share preview style surfaces,
- and temporary presentation forms that wrap existing content or actions.

Current entry points:

- `docs/dev/ui/foundations/transitions.md`
- `docs/dev/ui/foundations/animation.md`
- `docs/dev/ui/system-experiences/capture-and-share/print.md`
- `docs/dev/ui/system-experiences/capture-and-share/print-preview.md`
- `docs/dev/ui/system-experiences/capture-and-share/screencap-share.md`

Boundary note:

- the visible widget may be a component,
- but showing/dismissing/stacking behavior belongs here.
