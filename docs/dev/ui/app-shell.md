# App Shell

The app shell defines common surfaces used by apps:

- top-level navigation,
- command surfaces (search/command palette),
- modal and sheet conventions,
- consistent spacing, typography, and focus behavior.

Recommended patterns (zones, sidebars/inspector, desktop vs tablet windowing):

- `docs/dev/ui/app-shell-patterns.md`
